Linux （鸟哥）ShellScript

一、HelloWorld

  • 脚本使用外部的指令和bash，因此常呼叫外部的函数库，速度较慢，且使用的CPU资源较多。因此适合系统管理，但在大型数据处理上就不够好了

  • 编写注意事项：从上到下，从左到右，多个空白忽略，Tab视为空格，读到Enter就执行该行，#做注释

  • Script的执行方式：

    • 直接执行：绝对路径/相对路径；放到${PATH}；bash/sh xxx

    • source或.执行：在父程序中执行

    • 【注】使用新的bash（子程序）环境执行脚本，执行完后变量不会传入父程序；.\shell.sh需+x，sh shell.sh只需可读

#!/bin/bash
#宣告script使用的shell，该文件语法也需用bash语法。执行时加载bash的相关环境配置（一般是~/.bashrc）
PATH=/bin:/sbin:/usr/bin:/usr/sbin:/usr/local/bin:/usr/local/sbin:~/bin
export PATH  #环境变量宣告
echo -e 'Hello World! \a'  #发出警报声
exit 0  #执行成果返回（$?）

二、条件判断

1.test

#!/bin/bash
PATH=/bin:/sbin:/usr/bin:/usr/sbin:/usr/local/bin:/usr/local/sbin:~/bin
export PATH
filename='./xxx'
test ! -e $filename && echo "File '$filename' do not exist!" && exit 0
test -f $filename && filetype='file'
test -d $filename && filetype='directory'
echo "file '$filename' is a $filetype'"
exit 0

2.判断符号[]

中括号每个组件都需空格分隔，变量最好带双引号括号，常量最好带单或双引号

#!/bin/bash
PATH=/bin:/sbin:/usr/bin:/usr/sbin:/usr/local/bin:/usr/local/sbin:~/bin
export PATH
read -p 'Input Y/N: ' str
[ "${str}" == 'y' -o "${str}" == 'Y' ] && echo 'Yes!' && exit 0
# 也可多个`[]`隔开使用，其中`&&`表示and，`||`表示or
[ "${str}" == 'n' ] || [ "${str}" == 'N' ] && echo 'No!' && exit 0
echo 'Wrong Input!' && exit 0

3.判断式if

if [ ... ]; then
  ...
elif [ ... ]; then
  ...
else
  ...
fi
#!/bin/bash
PATH=/bin:/sbin:/usr/bin:/usr/sbin:/usr/local/bin:/usr/local/sbin:~/bin
export PATH
filename='./xxxx'
netstat -nult > $filename
if [ "$(grep ':21' $filename)" ];then
  echo 'FTP is on'
fi
if [ "$(grep ':22' $filename)" ];then
  echo 'SSH is on'
fi
if [ "$(grep ':80' $filename)" ];then
  echo 'WWW is on'
fi
#!/bin/bash
PATH=/bin:/sbin:/usr/bin:/usr/sbin:/usr/local/bin:/usr/local/sbin:~/bin
export PATH
read -p "Input you birthday month and day(eg.1025): " in
if [ ! $(echo $in | grep '[0-9]\{4\}') ];then
  echo "Wrong Input!"
  exit 1
fi
now=$(date +%m%d)
year=$(date +%Y)
now_s=$(date --date="$year$now" +%s)
if [ $now == $in ]; then
  echo "Happy Birthday!"
  exit 0
fi
if [ $now -gt $in ]; then
  year=$(($year+1))
fi
bir_s=$(date --date="$year$in" +%s)
res_s=$(($bir_s-$now_s))
res_d=$(($res_s/60/60/24))
echo "Your birthday will be $res_d days later."

三、函数

1.指令参数

command opt1 opt2 opt3
  $0     $1   $2   $3
$#：参数个数（3）
$@：表示"$1" "$2" "$3"
$*：表示"$1c$2c$3"，c为分隔符，默认为空格
# $4 == ''

shift：变量参数号码偏移

#!/bin/bash
PATH=/bin:/sbin:/usr/bin:/usr/sbin:/usr/local/bin:/usr/local/sbin:~/bin
export PATH
echo "$@---$1"
shift
echo "$@---$1"
shift 2
echo "$@---$1"
# liubin@localhost: ~$ ./hello.sh aa bb cc dd ee ff
# aa bb cc dd ee ff---aa
# bb cc dd ee ff---bb
# dd ee ff---dd

四、循环

1.普通循环

while [ ... ]  #成立时才执行
do
  ...
done
until [ ... ]  #不成立时才执行
do
  ...
done
#!/bin/bash
PATH=/bin:/sbin:/usr/bin:/usr/sbin:/usr/local/bin:/usr/local/sbin:~/bin

2.固定循环

for var in ...
do
  ...
done
#!/bin/bash
PATH=/bin:/sbin:/usr/bin:/usr/sbin:/usr/local/bin:/usr/local/sbin:~/bin
export PATH
# eg1
for x in dog cat mouse
do
  echo "-> $x"
done
# eg2
users=$(cut -d ":" -f1 /etc/passwd)
for x in $users
do
  id $x
done
# eg3
net="192.168.100"
for x in $(seq 70 80)  #{1..100}/{a..z}
do
  echo -n "$net.$x ->"  #-n不断行继续显示
  ping -c 1 -w 1 $net.$x &> /dev/null && echo "Connected" || echo "Disconneced"
done
#!/bin/bash
PATH=/bin:/sbin:/usr/bin:/usr/sbin:/usr/local/bin:/usr/local/sbin:~/bin
export PATH
read -p "Input a directory: " dir
if [ "$dir" == "" -o ! -d "$dir" ]; then
  echo "Directory $dir don't exist!"
  exit 1
fi
files=$(ls $dir)
for f in $files
do
  perm=""
  test -r "$dir/$f" && perm=$perm"r"
  test -w "$dir/$f" && perm=$perm"w"
  test -x "$dir/$f" && perm=$perm"x"
  echo "File $dir/$f's permission is [$perm]."
done

另一种方式：

for (( 初始; 判断; 运算 ))
do
  ...
done
#!/bin/bash
PATH=/bin:/sbin:/usr/bin:/usr/sbin:/usr/local/bin:/usr/local/sbin:~/bin
export PATH
read -p "Input a num: " in
sum=0
for (( i=1; i<=$in; i=i+1 ))
do
  sum=$(($sum+$i))
done
echo "1+2+...+$in = $sum"
